# Prashant Gupta - Data Professional Portfolio 🚀

This is the interactive, 3D portfolio website for **Prashant Gupta**, a Data Engineer, Data Analyst, and PG-DBDA (CDAC Mumbai). 
Check out the live website here: [https://prashant-gtx.github.io](https://prashant-gtx.github.io)

## 🌟 Overview

The portfolio is built with modern web technologies, featuring highly interactive 3D elements powered by ThreeJS and smooth scrolling/animations powered by GSAP.

## 🛠️ Techstack

- **Frontend Framework**: React, TypeScript, HTML, CSS, JavaScript
- **3D Rendering**: ThreeJS, WebGL, React Three Fiber, React Three Drei
- **Animations**: GSAP (GreenSock Animation Platform)

## 🚀 Setup & Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/prashant-gtx/prashant-gtx.github.io.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

## 📝 Note on GSAP Plugins
This project uses trial GSAP Club plugins for development. To use them in a commercial production environment, you may need a valid GSAP Club license. Check out the setup instructions here: [GSAP Installation Docs](https://gsap.com/docs/v3/Installation/)
